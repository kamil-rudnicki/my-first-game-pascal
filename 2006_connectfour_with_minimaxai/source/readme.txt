To compile this project in Microsoft Visual 2005:
1. Install wxWidgets 2.6.0
2. Set WXWIN path where you have wxWidgets
3. Compile wxWidgets with Multi-Threaded (/MT) option, Release !!!
4. Now you can compile Connect four (only release)
